Beta-Oxidation of Polyunsaturated Fatty Acids
       

It should be noted, that all polyunsaturated fatty acids with a divinyl-methyl configuration are oxidized using the following pathway. Arachadonic Acid is used only as a typical example for illustration purposes.

 
Arachidonyl-CoA (20:4cD5,8,11,14)
3,6,9,12-Octadecatetraenoyl-CoA (18:4cD3,6,9,12)
2,6,9,12-Octadecatetraenoyl-CoA (18:4D2,6,9,12)
4,7,10-Hexadecatrienoyl-CoA (16:3cD4,7,10)
2,4,7,10-Hexadecatetraenoyl-CoA (16:4D2,4,7,10)
3,7,10-Hexadecatrienoyl-CoA (16:3cD3,7,10)
2,7,10-Hexadecatrienoyl-CoA (16:3D2,7,10)

3,6-Dodecadienoyl-CoA (12:2cD3,6)
